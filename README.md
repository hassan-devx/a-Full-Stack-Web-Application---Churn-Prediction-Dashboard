# Churn Prediction Dashboard (Full-Stack Python Project)

## Overview
This project is a full-stack web application that predicts customer churn using machine learning and presents insights through an interactive dashboard.

## Features
- Customer churn prediction using ML model
- Data preprocessing and analysis
- Interactive dashboard for visualisation
- User-friendly interface

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- (Add Flask / Streamlit if used)

## Purpose
The goal of this project is to help businesses identify customers likely to leave and take proactive action.

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the application
